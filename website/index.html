How to install Step-by-step instructions ↓

Because this extension ships outside the Chrome Web Store, you install it in Developer Mode — a standard, safe, built-in browser feature supported by all Chromium browsers. This is how developers and power-users have always sideloaded trusted extensions.

1

Download & unzip

Click the button above. Save and unzip the file to a permanent folder — Chrome needs the folder to stay there.

2

Enable Developer Mode

Open your browser's extension manager: chrome://extensions (Chrome/Brave/Opera) or edge://extensions (Edge). Toggle Developer mode on — it's the switch in the top-right corner.

3

Click "Load unpacked"

Click Load unpacked and select the unzipped folder. FactChecker AI will appear instantly.

4

Add your API key

Click the extension icon → open Settings → paste your free Gemini API key. You're done.
